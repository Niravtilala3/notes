Overall Layout & Navigation

How should users navigate between boards, tags, and tools?

A

Sidebar + Canvas (Classic)

Persistent left sidebar for boards, tags, and settings. Infinite canvas on the right.

B

Floating UI (Immersive)

Canvas takes up the whole screen. Tools are in a floating dock or accessible via Command Palette.

C

Top Bar + Canvas (Simple)

Standard top navigation bar for switching boards, leaving maximum vertical space for the canvas.
